# AWS Lambda .NET App Development

## About
- This folder contains multiple standalone AWS Lambda .NET applications written in c# programming language
- The version of the .NET applications will range from .NET 5 to .NET 10

## Workflow
1. **Research**: Create `2-RESEARCH.md` implementation plan from `1-TODO-CHSARP-LAMBDA.md`
2. **Review**: Wait for user approval of `2-RESEARCH.md`
3. **Plan**: Create detailed `3-PLAN.md` with atomic steps from `2-RESEARCH.md`
4. **Review**: Wait for user approval of `3-PLAN.md`
4. **Implement**: Execute step-by-step of `3-PLAN.md`, mark "(COMPLETED)"

## Do
- Assume no prior dev knowledge
- Small, atomic steps
- Individual tests
- Wait for explicit approval between workflow
- Keep the setup and code as simple as possible
