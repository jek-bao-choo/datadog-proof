# Note
* Always use terminal CLI with NPM to create a web app via Vite. Vite is a build tool like Webpack.

## Setup steps below

Setup a web app using Vite:
```
npm create vite@latest
```

```
After which follow the Interactive Prompts:
The command will prompt you with a few questions to configure your project.

Project name: Enter the name for your project's folder. For example: my-vue-app

✔ Project name: … my-vue-app
Select a framework: Use the arrow keys to navigate the list and press Enter. Choose vue.

✔ Select a framework: › - Use arrow keys. Return to submit.
    vanilla
❯   vue
    react
    preact
    lit
    svelte
    solid
    qwik
    others
Select a variant: This allows you to choose between standard JavaScript or TypeScript, and offers further customization.

Vue: For a standard JavaScript setup.

TypeScript: For a TypeScript setup.

Customize with create-vue...: This is the most powerful option, allowing you to add things like Vue Router, Pinia for state management, Vitest for testing, ESLint, and Prettier right from the start.
```
